# TypeScriptProject01


## **Create new index.ts**

- **`Intall type script`**

```
npm install -g typescript
```

- **`Fix the error`**

```
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

- **`Run typeScript file`**

```
tsc index.ts
```

- **`Run javaScript file`**

```
node index.js
```

- **`Init project file`**

```
tsc --init
```

- **`Refactor project and run`**

```
tsc
```
